tcp_reassembly

This module provides the functionality of stream reassembly.

The input SHOULD be a pcap dump file relative to just one stream: alternatively it's up to the user to pass the correct pcap_filter string in order to filter out the one stream needed.

In addition the program can be launched as a command line utility.
